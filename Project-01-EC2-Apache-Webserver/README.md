# Project 01: EC2 Apache Web Server

## Description
This project demonstrates launching an EC2 instance on AWS, installing Apache Web Server, and hosting a static webpage accessible via the public IP.

## AWS Services Used
- IAM
- EC2
- Security Groups

## Steps Performed
1. Launched EC2 using Amazon Linux 2
2. Configured Security Group to allow SSH and HTTP
3. Connected to EC2 via SSH
4. Installed and started Apache Web Server
5. Created a static HTML page
6. Accessed the webpage using EC2 public IP

Refer to `commands.txt`

## Output
Webpage successfully accessed via browser.
